# /rwa-lifecycle

Use this command when the user wants a step-by-step lifecycle review for an RWA product.

## Suggested workflow

1. Clarify the asset being tokenized
2. Identify issuer, manager, custodian, and compliance roles
3. Map on-chain controls to off-chain processes
4. Review issuance, yield, redemption, and transfer steps
5. Surface risks, assumptions, and open questions
